# 多言語対応クイックスタート

## 🚀 3ステップで多言語対応

### ステップ1: LocalizedButton を配置

1. シーンツリーで「+」ボタンをクリック
2. 「LocalizedButton」で検索
3. ノードを追加

![ノード追加](画像予定)

---

### ステップ2: 翻訳キーを設定

インスペクターで `Translation Key` に翻訳キーを入力

```
Translation Key: Title.start_game
```

![インスペクター設定](画像予定)

**エディタ上の表示**: `[Title.start_game]`

---

### ステップ3: JSONファイルに翻訳を追加

#### 日本語 (`resources/localization/ja.json`)

```json
{
  "Title": {
    "start_game": "ゲームスタート"
  }
}
```

#### 英語 (`resources/localization/en.json`)

```json
{
  "Title": {
    "start_game": "Start Game"
  }
}
```

---

## ✅ 完了！

ゲームを実行すると:
- 日本語設定: 「ゲームスタート」と表示
- 英語設定: 「Start Game」と表示

**スクリプトは一切不要です！**

#### JSON (`en.json`)
```json
{
  "Title": {
    "start_game": "Start Game"
  },
  "Settings": {
    "title": "Settings",
    "language": "Language",
    "volume": "Volume"
  }
}
```

#### シーンで使用
```
LocalizedLabel
└── Translation Key: Settings.title
```

## 💡 TIPS

### パラメータを使う

**スロット番号を表示:**

```json
{
  "Save": {
    "slot": "スロット {0}"
  }
}
```

スクリプトで:
```gdscript
@onready var slot_label: LocalizedLabel = $SlotLabel

func _ready() -> void:
    slot_label.translation_key = "Save.slot"
    slot_label.translation_params = [3]  # "スロット 3"
```

---

## 🔧 言語を切り替える

設定画面などで:

```gdscript
# 英語に切り替え
LocalizationManager.change_language("en")

# → すべての LocalizedUI が自動的に英語に！
```
