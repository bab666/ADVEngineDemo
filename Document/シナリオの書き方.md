## シナリオの書き方

### ファイル形式

- UTF-8テキスト
- 拡張子: `.txt`
- 場所: `resources/scenarios/`

### 基本文法

#### コメント

```
# これはコメント行（無視される）
```

#### 背景切り替え

```
@bg room
@bg forest
```

#### キャラクター表示

```
@chara alice normal 1920 540
@chara bob smile 960 540
```

#### キャラクター非表示

```
@chara_hide alice
```

#### 台詞

```
主人公:おはよう。
アリス:おはようございます！
```

#### BGM再生

```
@bgm file=peaceful_day
@bgm file=tension volume=80 loop=true
```

#### BGM停止

```
@stopbgm time=2.0
```

#### 待機

```
@wait time=2000
```

#### ウインドウ切り替え

```
@window default
主人公:これは通常のウインドウです。

@window sub
ナレーター:これはサブウインドウです。
```

### サンプルシナリオ

```
# デモシナリオ
@bg room
@bgm file=peaceful_day volume=80

@chara alice normal 1920 540
主人公:今日はいい天気だ。

@chara alice smile 1920 540
アリス:そうですね！散歩に行きませんか？

主人公:いいね、行こう。

@wait time=1000
@chara_hide alice

# シーン転換
@stopbgm time=2.0
@bg forest
@bgm file=tension volume=90

主人公:...なんだか怪しい雰囲気だ。
```
