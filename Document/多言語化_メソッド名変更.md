# メソッド名変更の重要なお知らせ

## 変更内容

`LocalizationManager` のメソッド名を変更しました。

### 変更理由

Godot の `Object` クラスには既に `tr()` という組み込みメソッドが存在するため、これをオーバーライドするとエラーが発生します。

### 変更前後

**変更前:**
```gdscript
LocalizationManager.tr("Title.start_game")
```

**変更後:**
```gdscript
LocalizationManager.get_text("Title.start_game")
```

## 全APIリファレンス

### 基本的な使用方法

```gdscript
# 翻訳テキストを取得
var text: String = LocalizationManager.get_text("Title.start_game")
# -> "ゲームスタート" (日本語) or "Start Game" (英語)

# パラメータ付き翻訳
var text: String = LocalizationManager.get_text("Save.slot", [1])
# -> "スロット 1" or "Slot 1"

var text: String = LocalizationManager.get_text("Common.page", [3, 10])
# -> "ページ 3/10" or "Page 3/10"
```

### その他のメソッド（変更なし）

```gdscript
# 言語を英語に変更
LocalizationManager.change_language("en")

# 現在の言語を取得
var current: String = LocalizationManager.get_current_language()

# 言語の表示名を取得
var name: String = LocalizationManager.get_current_language_name()

# 言語変更を監視
LocalizationManager.language_changed.connect(_on_language_changed)
```

---

## 既存コードの修正が必要な箇所

すべての `LocalizationManager.tr()` を `LocalizationManager.get_text()` に置き換えてください。

**検索と置換:**
- 検索: `LocalizationManager.tr(`
- 置換: `LocalizationManager.get_text(`
